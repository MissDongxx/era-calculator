What Is ERA?

ERA means Earned Run Average(ERA). It estimates how many earned runs a pitcher would allow across one official game length.

Earned runs only

ERA uses earned runs, not every run on the scoreboard. Runs that score mainly because of fielding errors are usually treated as unearned and are not included in ERA.

Game length matters

Baseball ERA usually uses 9 innings. Many softball and youth formats use 7 innings. The calculator lets you set the official game length before calculating.

Partial innings

Scorebook notation uses .1 for one out and .2 for two outs. Those are thirds of an inning, so 6.1 innings means 6 and 1/3 innings.

Basic ERA rules

A lower ERA is better because it means fewer earned runs allowed per game. ERA is most useful when comparing pitchers over meaningful innings totals; short relief outings can produce extreme results from just one run. For season ERA, add total earned runs and total innings pitched first, then calculate once from the totals.

ERA Calculator Guide

The ERA calculator turns earned runs and innings pitched into a single Earned Run Average(ERA) result for an outing, tournament, or season.

What it does

It calculates how many earned runs a pitcher allows per official game length. The result is a rate, so pitchers with different workloads can be compared more fairly.

Required inputs

Use earned runs, full innings pitched, extra outs, and game length. The tool separates innings from outs so scorebook notation is handled correctly.

Reading the result

A 3.00 ERA means three earned runs allowed per official game length. It does not mean the pitcher allowed exactly three runs in every appearance.

How to Calculate ERA

To calculate ERA manually, multiply earned runs by the official game length, then divide by innings pitched.

Step-by-step method

First, confirm the earned runs allowed. Second, convert innings pitched into a true innings value. Third, multiply earned runs by game length. Finally, divide by innings pitched.

ERA = earned runs x game length / innings pitched

If a pitcher allows 2 earned runs in 6 innings and 1 out during a 9-inning game, the innings value is 6.333. The ERA is 2 x 9 / 6.333 = 2.84.

Common mistake: 6.1 means six innings and one out, not 6.1 decimal innings.

ERA Calculator FAQ

What is a good ERA?

Context matters, but lower is better. In many baseball environments, an ERA below 3.00 is strong, around 4.00 is serviceable, and higher values need league context.

How do partial innings work?

One out is one-third of an inning and two outs are two-thirds. A line of 6.1 means six innings and one out, not 6.1 decimal innings.

Can I use this for softball?

Yes. Change game innings to 7 for common softball formats, or use your league's official game length.
