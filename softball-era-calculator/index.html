Softball ERA Calculator

The softball ERA calculator is designed for softball pitching lines, especially common 7-inning formats.

Softball game length

Many softball leagues use 7 innings for ERA. If your league uses another official game length, use that number instead.

Softball calculation

For a 7-inning game, the calculation is earned runs x 7 / innings pitched.

Example

A pitcher allows 1 earned run in 4 innings during a 7-inning softball game. The ERA is 1 x 7 / 4 = 1.75.

Use the calculator

Open the ERA calculator, set game innings to 7, and enter the softball pitching line.
