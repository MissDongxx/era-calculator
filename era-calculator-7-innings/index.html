ERA Calculator 7 Innings

An ERA calculator 7 innings page is for leagues where the official game length is 7 innings instead of 9.

7-inning formula

ERA = earned runs x 7 / innings pitched. The calculation changes only the game-length multiplier.

7-inning example

If a pitcher allows 2 earned runs in 5 innings during a 7-inning game, the ERA is 2 x 7 / 5 = 2.80.

Partial innings

Use one-third for one out and two-thirds for two outs before applying the 7-inning calculation.

Use the calculator

Open the ERA calculator, set game innings to 7, and enter the pitching line.
